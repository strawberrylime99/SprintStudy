# SprintStudy Starter Pack (Pre-Bundle)

This folder contains starter files you can customize into your final paid bundle.

## Included files

- `notion/Assignments-Database.csv`
- `notion/Weekly-Plan-Board.csv`
- `sheets/Assignment-Tracker.csv`
- `sheets/Grade-Target-Calculator.csv`
- `printables/Weekly-Checklist.html`

## How to turn this into a final product

1. Import the Notion CSV files into your Notion workspace as databases.
2. In Google Sheets or Excel, import both CSV sheet files and adapt formulas to your preferred layout.
3. Open `Weekly-Checklist.html` in a browser and Print to PDF.
4. Add branding (logo, colors, examples).
5. Export final bundle files for Etsy delivery.

## Recommended final bundle structure

- `SprintStudy_Notion_Template.pdf` (shared template instructions + duplicated workspace link)
- `SprintStudy_Tracker.xlsx` and `SprintStudy_Tracker_GoogleSheets_Link.txt`
- `SprintStudy_Weekly_Checklist.pdf`
- `SprintStudy_Quick_Start_Guide.pdf`
