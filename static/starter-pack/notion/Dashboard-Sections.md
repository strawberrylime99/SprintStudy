# Dashboard Sections (Paste Into Your Main Notion Page)

## 1) This Week Sprint
- Linked database view: Weekly Sprints
- Filter: `Week` contains current week
- Show: Primary Outcome, Must Do 1-3, Buffer Block, Status

## 2) Due Soon (Manual view build)
- Linked database view: Assignments
- Suggested filter: `Due Date` is within next 7 days AND `Status` is not Submitted
- Sort: `Due Date` ascending, `Priority` descending

## 3) Grade Target Snapshot
- Linked database view: Assignments
- Show columns: Grade Weight %, Target Score %, Actual Score %, Points Earned, Points Possible

## 4) Study Blocks This Week
- Linked database view: Study Blocks
- Filter: current week
- Sort: Day ascending then Start Time ascending

## 5) Weekly Reflection
- Text block prompt:
  - What moved me forward this week?
  - What bottleneck repeated?
  - What will I remove or simplify next week?
